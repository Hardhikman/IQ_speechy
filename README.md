# IQ Speechy

This repository contains the IQ Speechy project, a text-to-speech (TTS) system using the NeuTTS Air library.

## Features

- Text-to-speech conversion with voice cloning capabilities
- Support for multiple preset speakers (Dave, Jo)
- Custom voice cloning with reference audio
- Web interface for easy interaction

## Getting Started

1. Clone the repository
2. Install the required dependencies
3. Run the application

## Dependencies

- Python 3.11+
- neuttsair
- soundfile
- torch
- numpy
- gradio

## Usage

Run the main application:
```bash
python app.py
```

Then open your browser to http://localhost:7860 to access the web interface.
